Lesson 1: Webpages, Documents and Structure

1. HTML and CSS are both languages

HTML - basic tool is a tag - turns into elements on a tree
CSS - changes how elements look on a page

2. HTML Controls the structure of the page

It has a tree-like structure. The basic part is the "tag" which creates an element. A tag can have attributes with values.

It is a good practice to use div tags to organize similar sections within a document. This will allow you to set up style rules that cn be applied universally and reduce repetition of the same code.

3. CSS Controls the "Style" of the page.

Stands for Cascading Style Sheets. It is done by giving similar HTML elements the same class name and then specifiing the style that should apply. It saves you from having to repeat the same code for similar elements on the page. Avoiding repetition is simpler and makes it easier to make changes universally to similar elements.

4. When Programmers talk about the DOM - Document Object Model - they are talking about the structure of a page.

It shows elements on the page and how they relate to each other.

Python Variables

Variables give programmers a way to give names to their variables.

For example:
s[s] equals s[1+1+1]
s[0] equals (s+s)[0]
s[-1] equals (s+s)[-1]

Assigning Value to a Variable

You can change the value of a variable by reassigning it to different values later.
For example: if you have the string be named 'word' which is assigned to be assume:

print word[3] equals u
print word[3:4] equals u
print word[3:3] equals blank - nothing is inbetween location 3 and itself
print word[4:] equals me
print word[4:6] equals me
print word[:2] equals as
print word[:] equals assume

Understanding equals signs

In the example of 2+3=5, the = is "same as".
In the example of "my_variable=5", the = means "takes the value of"

Ways variable are useful

Variables improve readability by using names that make sense to people. It gives us a way to store the value of important data. It gives us a way to change the value of something.
